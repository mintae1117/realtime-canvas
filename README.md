# 실시간 학습 캔버스 (Realtime Learning Canvas)

React Konva와 Supabase Realtime을 사용한 실시간 협업 캔버스 애플리케이션입니다. 선생님과 학생이 함께 문제를 풀고 학습할 수 있는 플랫폼입니다.
방문하기 : [링크](https://realtime-canvas-gray.vercel.app/)


## 주요 기능

- ✏️ **실시간 그리기**: 펜, 지우개, 선택 도구를 사용한 자유로운 필기
- 📷 **이미지 업로드**: 문제 이미지나 참고 자료를 캔버스에 추가
- 👥 **다중 사용자**: 여러 사용자가 동시에 같은 캔버스에서 작업
- 🔄 **실시간 동기화**: Supabase Realtime을 통한 즉각적인 변경사항 반영
- 🎨 **다양한 도구**: 색상 선택, 선 굵기 조절, 텍스트 추가
- 👨‍🏫 **역할 구분**: 선생님/학생 역할 선택
- 🌈 **사용자 식별**: 각 사용자에게 고유한 색상 부여

## 기술 스택

- **Frontend Framework**: React 19 + TypeScript
- **Canvas Library**: React Konva
- **State Management**: Zustand
- **Realtime**: Supabase Realtime
- **Styling**: Tailwind CSS
- **Build Tool**: Vite


## 사용 방법

1. **방 입장**

   - 이름을 입력하세요
   - 역할(선생님/학생)을 선택하세요
   - 방 ID를 입력하거나 새로 생성하세요
   - 같은 방 ID를 사용하면 함께 학습할 수 있습니다

2. **그리기**

   - 펜 도구를 선택하고 캔버스에 그리세요
   - 색상과 선 굵기를 조절할 수 있습니다
   - 지우개로 실수한 부분을 지울 수 있습니다

3. **이미지 추가**

   - "이미지 업로드" 버튼을 클릭하세요
   - 문제 이미지나 참고 자료를 선택하세요
   - 이미지가 캔버스에 추가됩니다

4. **협업**
   - 모든 참가자가 실시간으로 서로의 작업을 볼 수 있습니다
   - 참가자 목록에서 현재 접속 중인 사용자를 확인할 수 있습니다

## 주요 컴포넌트 설명

### Canvas.tsx

React Konva를 사용한 메인 캔버스 컴포넌트입니다. 그리기, 이미지 표시, 텍스트 렌더링 등의 기능을 담당합니다.

### useRealtimeCanvas.ts

Supabase Realtime을 통한 실시간 동기화를 처리하는 커스텀 훅입니다. 브로드캐스트와 프레즌스 기능을 사용합니다.

### canvasStore.ts

Zustand를 사용한 전역 상태 관리 스토어입니다. 캔버스 데이터, 사용자 정보, 도구 상태 등을 관리합니다.
